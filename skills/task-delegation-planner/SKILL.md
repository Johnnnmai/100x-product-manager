# What you get

- human vs agent split
- task contracts
- context requirements

# Common failure

Teams delegate based on convenience instead of judging where human judgment is still required.

# Best for

- teams splitting work between humans and agents
- PMs defining execution contracts

# 3-5 key questions

1. Which work needs human judgment?
2. Which work is deterministic enough for an agent?
3. What context must each actor receive?
4. What should trigger escalation?

# Working logic

1. Separate judgment-heavy from execution-heavy work.
2. Assign agent work only where the contract can be explicit.
3. Define context, constraints, and handoff rules.

# Output format

- human-owned tasks
- agent-owned tasks
- context needed
- escalation triggers

# Review checklist

- Is the split based on judgment, not preference?
- Are agent contracts explicit?
- Are escalation triggers visible?
