# 100x Growth Sprint / 100倍增长冲刺

[English](#english) | [中文](#中文)

---

## English

### When to Use This Skill

Use this when you're ready to **execute a major growth initiative** that could 10x your business.

### Duration

**4 Weeks**

### Sprint Goal

Identify and execute on the highest-impact growth lever to achieve 10x growth potential.

### Week-by-Week Plan

---

#### Week 1: Identify the 10x Lever

**Monday-Wednesday:**
- Analyze current metrics
- Identify all growth levers
- Rank by potential impact

**Thursday-Friday:**
- Select top 1-2 levers
- Design experiments

**Output:**
- Top 10x lever(s) identified
- Experiment designs

---

#### Week 2: Build Fast

**Monday-Wednesday:**
- Build minimum test for each lever
- Use existing infrastructure
- Fake it if needed

**Thursday-Friday:**
- QA and prepare for launch
- Set up tracking

**Output:**
- Working experiments ready to run
- Success metrics defined

---

#### Week 3: Run Experiments

**Monday-Friday:**
- Launch experiments
- Monitor daily
- Gather data

**Daily Check:**
- Is the experiment working?
- Any unexpected results?
- Need to pivot?

**Output:**
- Experiment data
- Early learnings

---

#### Week 4: Analyze & Scale

**Monday-Wednesday:**
- Deep analysis of results
- Identify what worked

**Thursday-Friday:**
- Decision: Scale/Pivot/Stop
- Scale winning experiments
- Plan next phase

**Output:**
- Results analysis
- Scale plan

---

### Sprint Outputs

| Output | Description |
|--------|-------------|
| 10x Lever | The key growth lever identified |
| Experiments | 1-2 experiments run |
| Data | Validated growth hypothesis |
| Decision | Scale/Pivot/Stop |
| Next Steps | How to continue growth |

---

### Success Criteria

**Winning means:**
- [ ] Clear winner experiment identified
- [ ] Path to 10x demonstrated (not guaranteed, but possible)
- [ ] Clear next steps to scale

---

### Key Principle

**Think Big, Move Fast:**
- Don't optimize - transform
- One big bet > many small bets
- Speed reveals what matters

## 中文

### 何时使用此技能

当准备好**执行重大增长计划**可能使业务增长10倍时使用。

### 时长

**4周**

### 冲刺目标

识别并执行最高影响的增长杠杆，以实现10倍增长潜力。

### 周计划

---

#### 第一周：识别10倍杠杆

**周一至周三：**
- 分析当前指标
- 识别所有增长杠杆
- 按潜在影响排序

**周四至周五：**
- 选择前1-2个杠杆
- 设计实验

**输出：**
- 确定顶级10倍杠杆
- 实验设计

---

#### 第二周：快速构建

**周一至周三：**
- 为每个杠杆构建最小测试
- 使用现有基础设施
- 如需要可以伪造

**周四至周五：**
- QA并准备启动
- 设置追踪

**输出：**
- 准备运行的实验
- 定义成功指标

---

#### 第三周：运行实验

**周一至周五：**
- 启动实验
- 每日监控
- 收集数据

**每日检查：**
- 实验有效吗？
- 有意外结果吗？
- 需要转向吗？

**输出：**
- 实验数据
- 早期学习

---

#### 第四周：分析和规模化

**周一至周三：**
- 深入分析结果
- 识别什么有效

**周四至周五：**
- 决定：规模化/转向/停止
- 规模化有效的实验
- 计划下一阶段

**输出：**
- 结果分析
- 规模化计划

---

### 冲刺输出

| 输出 | 描述 |
|------|------|
| 10倍杠杆 | 识别的关键增长杠杆 |
| 实验 | 运行1-2个实验 |
| 数据 | 验证增长假设 |
| 决定 | 规模化/转向/停止 |
| 下一步 | 如何继续增长 |

---

### 成功标准

**成功意味着：**
- [ ] 明确识别出获胜实验
- [ ] 展示10倍路径（不保证，但可能）
- [ ] 明确规模化下一步

---

### 关键原则

**想大的，动快的：**
- 不要优化——要转型
- 一个大赌注 > 许多小赌注
- 速度揭示什么重要
