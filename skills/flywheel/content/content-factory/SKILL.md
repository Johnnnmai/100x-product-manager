# SKILL: content-factory

## Overview
Create distribution-ready content from product assets.

## When to Use
- Repurposing ebooks into social content
- Creating promotional materials
- Building content calendar
- Generating platform-specific variants

## Inputs
- Source asset (ebook, template, etc.)
- Target platform (LinkedIn, XHS, Newsletter, etc.)
- Content goals (awareness, engagement, conversion)

## Process

### 1. Hook Generation
- Extract key insights from asset
- Create multiple hook variations
- Test emotional triggers

### 2. Content Adaptation
- Platform-specific formatting
- Thread vs single post vs story
- Visual/text balance

### 3. Distribution Prep
- Add CTA variants
- Create engagement prompts
- Prepare follow-up sequences

## Outputs
- Content pieces (hooks, posts, threads)
- Platform-specific adaptations
- Scheduling recommendations
- Engagement metrics targets

## Content Types
- Hooks: Attention-grabbing openings
- Scripts: Video/audio outlines
- Posts: Long-form social content
- Threads: Multi-part narratives
