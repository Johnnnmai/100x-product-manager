# SKILL: experiment-tracker

## Overview
Design, track, and analyze experiments across the revenue flywheel.

## When to Use
- Testing new offers
- Optimizing conversion funnels
- Validating pricing hypotheses
- Measuring channel effectiveness

## Inputs
- Experiment hypothesis
- Control and variant definitions
- Success metrics
- Sample size requirements

## Process

### 1. Design
- Define null and alternative hypotheses
- Select primary and secondary metrics
- Calculate required sample size
- Set confidence thresholds

### 2. Execution
- Configure tracking (events, UTM params)
- Randomization setup
- Monitor for anomalies

### 3. Analysis
- Statistical significance testing
- Segment analysis
- Learnings extraction

## Outputs
- Experiment design document
- Tracking configuration
- Results dashboard
- Recommendations report

## Metrics Framework
- Signals: Impressions, reach
- Engagement: Clicks, time on page
- Conversion: Leads, purchases
- Retention: Repeat buyers, LTV
