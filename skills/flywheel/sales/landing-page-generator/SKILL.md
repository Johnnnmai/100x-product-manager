# SKILL: landing-page-generator

## Overview
Create high-converting landing pages for product offers.

## When to Use
- Launching new products
- Creating sales pages
- Building waitlists
- Testing offer positioning

## Inputs
- Product/offer details
- Target audience
- Conversion goals
- Brand guidelines

## Process

### 1. Structure
- Hero section with clear value prop
- Problem/agitation section
- Solution demonstration
- Social proof integration
- Call-to-action optimization

### 2. Copywriting
- Benefit-focused headlines
- Feature-translation to outcomes
- Urgency and scarcity elements
- Risk reversal (guarantees)

### 3. Technical
- Responsive design
- Fast loading
- Analytics setup
- A/B test points

## Outputs
- Complete landing page (Notion/HTML)
- Copy deck
- Analytics configuration
- A/B test recommendations

## Best Practices
- Above-fold CTA
- Minimal distractions
- Clear next steps
- Trust signals
