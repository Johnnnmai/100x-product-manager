# SKILL: ebook-factory

## Overview
Create digital assets (ebooks, templates, checklists) for product delivery.

## When to Use
- Building product catalog
- Creating lead magnets
- Developing course materials
- Producing template libraries

## Inputs
- Offer specification
- Target format (PDF, Notion, HTML, etc.)
- Delivery timeline
- Quality standards

## Process

### 1. Content Structure
- Define chapter/module outline
- Create supporting visuals
- Design interactive elements

### 2. Production
- Write core content
- Add examples and case studies
- Include actionable templates

### 3. Polish
- Edit for clarity and flow
- Add design elements
- Ensure print/digital compatibility

## Outputs
- Completed digital asset
- Source files
- Usage guide

## Product Tiers
- Cash Products ($9-39): Checklists, simple templates
- Flagship Products ($49-199): Comprehensive guides, playbooks
- Micro Assets ($300-2k): Complete systems, courses
