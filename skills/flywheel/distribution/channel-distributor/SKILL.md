# SKILL: channel-distributor

## Overview
Execute multi-channel content distribution and track performance.

## When to Use
- Launching new content
- Scaling distribution
- Testing new channels
- Optimizing posting schedules

## Inputs
- Content pieces ready for distribution
- Target channels and budgets
- Timing requirements
- Attribution needs

## Process

### 1. Channel Strategy
- Match content to channel strengths
- Optimize posting times
- Set up tracking links

### 2. Execution
- Schedule posts across platforms
- Configure automated sequences
- Set up retargeting rules

### 3. Monitoring
- Track engagement metrics
- Monitor conversion signals
- Identify viral potential

## Outputs
- Published content across channels
- Performance dashboard
- Optimization recommendations

## Channels
- Social: LinkedIn, X (Twitter), Xiaohongshu
- Email: Newsletter, sequences
- Organic: SEO, communities
- Paid: Retargeting, boost posts
