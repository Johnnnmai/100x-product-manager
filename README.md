## English

### Philosophy

**Move Fast. Iterate Fast. Market-Driven.**

100X Product Managers is a collection of high-impact PM skills that emphasize:

- 🎯 **Market-Driven**: Build what the market needs, not what we think
- ⚡ **Move Fast**: Speed is a competitive advantage
- 🔄 **Iterate Fast**: Learn fast, fail fast, pivot fast
- 📈 **10x Thinking**: Aim for 10x, not 10%

### Why This Exists

Traditional PM frameworks are often too process-heavy and internally-focused. They answer "how to do" but miss "when to do" and "what the market actually wants."

This skill collection takes a different approach:
- **Quality over quantity** - 8 strategic skills vs 46 generic ones
- **Speed as core competency** - Every skill emphasizes fast iteration
- **Market validation first** - Always ask "does the market need this?"

### Skills Overview

| Category | Skills | Description |
|----------|--------|-------------|
| **Core** | 3 | Foundational thinking frameworks |
| **Interactive** | 3 | Guided discovery with adaptive questions |
| **Workflow** | 2 | End-to-end PM processes |

#### Core Skills (3)

1. **market-driven-prioritization** - Market-driven prioritization framework
   - Core question: "Does the market really need this?"
   - Output: Priority list with market validation status

2. **rapid-iteration-cycle** - Fast iteration cycle
   - Core question: "How to validate an idea in 2 weeks?"
   - Output: 2-week MVP plan + validation metrics

3. **10x-hypothesis-framework** - 10x growth hypothesis framework
   - Core question: "What change can bring 10x growth?"
   - Output: 10x growth hypothesis + experiment roadmap

#### Interactive Skills (3)

4. **market-signal-detector** - Market signal detection
5. **launch-decision-engine** - Launch decision engine
6. **competitive-move-generator** - Competitive response generator

#### Workflow Skills (2)

7. **market-validation-sprint** - Market validation sprint (1 week)
8. **100x-growth-sprint** - 100x growth sprint (4 weeks)

## 中文

### 核心理念

**快速行动，快速迭代，市场驱动**

100X Product Managers 是一套高影响力的 PM 技能集合，强调：

- 🎯 **市场驱动**: 构建市场需要的，而非我们认为的
- ⚡ **快速行动**: 速度是核心竞争力
- 🔄 **快速迭代**: 快速学习，快速失败，快速转向
- 📈 **10倍思维**: 瞄准10倍增长，而非10%

### 为什么存在

传统的 PM 框架往往过于流程化、过于内部导向。它们回答"如何做"，却忽略了"何时做"和"市场真正需要什么"。

这个技能集合采用不同的方法：
- **质量优先于数量** - 8个战略性技能 vs 46个通用技能
- **速度作为核心能力** - 每个技能都强调快速迭代
- **市场验证优先** - 始终问"市场需要这个吗？"
