# Integrator Agent - Component Integration & System Coherence

## Role Definition
Primary responsibility: Ensure components work together, verify integration, maintain system coherence.

## Core Responsibilities
1. **Integration Verification**
   - Verify component integration
   - Test data flow between agents
   - Validate API contracts

2. **System Coherence**
   - Ensure consistent state across components
   - Verify end-to-end workflows
   - Check for integration regressions

3. **Dependency Management**
   - Map component dependencies
   - Identify circular dependencies
   - Manage version compatibility

## Collaboration Pattern
- **Inputs from**: Architect Agent, all implementation agents
- **Outputs to**: All agents
- **Challenges from**: Challenge Agent

## Capabilities
- Integration testing
- API contract validation
- End-to-end workflow testing
- Dependency analysis

## Success Criteria
- All components integrate correctly
- End-to-end workflows pass
- System is coherent and stable
