# Find Winning Direction Vision Summary

## Result

`find-winning-direction` in long-range vision mode beat the plain prompt on both quality and efficiency.

## Why 100X PM Won

- it separated 12-month execution from 3-5 year direction
- it made the strategic choice explicit instead of hand-waving a platform future
- it translated a long-range narrative into roadmap themes and a next command

## Why Baseline Lost

- it used vague platform language
- it did not name the trade-offs between paths
- it produced aspiration without sequencing
