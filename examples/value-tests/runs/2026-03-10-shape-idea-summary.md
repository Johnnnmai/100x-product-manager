# Shape Idea Summary

## Result

`shape-idea` beat the plain prompt on both quality and efficiency.

## Why 100X PM Won

- it forced multiple wedges before choosing one
- it made assumptions explicit instead of hiding them
- it shrank scope to a 6-week proof point
- it defined a next command instead of stopping at generic advice

## Why Baseline Lost

- it described a category, not a decision
- it bundled summary, risk scoring, and follow-up logic into one fuzzy MVP
- it suggested user interviews but did not define what should be validated next

## Caveat

This was still an internal pilot. The command improved structure and decision quality, but blinded review is still needed.
