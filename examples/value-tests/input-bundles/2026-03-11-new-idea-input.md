# 2026-03-11 New Idea Input Bundle

## Scenario

A founder has a rough product idea and wants a direction worth betting on.

## Idea

"I want to build a product OS for founders and PMs that uses social signals and user simulation to validate ideas before teams start building."

## Founder Instinct

- founders waste time because they jump from vague idea to build
- PMs sit in meetings with too much context and weak decisions
- AI should help reduce ambiguity before engineering work starts

## Constraints

- solo founder plus one contract engineer
- needs a convincing proof point in 6 weeks
- cannot build a giant multi-agent platform first
- must be understandable in one demo

## Success Criteria

- choose a wedge
- define the business model
- choose the first channel
- specify what should be validated next
